## What is SPN?

SPN (Service Provider Name) is a text field defined in the eSIM profile. The device OS uses the value specified in it to display on certain screens during interaction with the phone — for example, the eSIM install screen.

## What is SPN customisation?

SPN customisation means the customer can specify any value in it, and Bondio will deliver all their eSIMs with this specific configuration.

## How do I get a customised SPN?

1. Provide Bondio the SPN value over chat/email
2. Bondio will then generate a customised eSIM profile. This will add a new entry in your eSIM inventory table on the Bondio web portal. Generating a customised eSIM profile can take up to **3–4 working days**.

## How is the SPN value handled on Android?

On Android, the SPN value is shown:

1. On the eSIM install screen
2. Beside the cellular signal bars in the top status bar of the screen — however, this is not true for all mobile handsets

## How is the SPN value handled on iOS?

Unfortunately, iOS doesn't use the SPN value visibly anywhere as far as we know. Apple has its own logic to determine the values shown on different screens like the eSIM install screen.

<Note>
For iOS, in order to show your branding you need to be using Apple privileged eSIMs — otherwise this feature can't be implemented.
</Note>

## How long does it take to customise the SPN?

3–4 working days.

## How much does it cost?

One-time fixed cost per eSIM label. For the exact amount, please reach out to your Bondio point of contact.

For example, if you want to customise the SPN for Tau & Lambda eSIMs on the Bondio platform, that counts as 2 customisations.

## Does it change the SM-DP+ address?

The SPN customisation doesn't change the domain name `rsp.bondio.co` in the SM-DP+ address.
