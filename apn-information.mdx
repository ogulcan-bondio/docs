## APN configuration details

We use **auto-APN configuration**, which means the APN field is automatically populated once the eSIM is downloaded.

### Auto-APN feature

All our eSIMs use automatic APN configuration:

- APN field is automatically populated upon eSIM download
- No user action required
- No configuration needed from clients or end-users

<Note>
Each label has a different APN, so please make sure to check the ICCID for specific information. This can now be found in the plan details on the dashboard.
</Note>
